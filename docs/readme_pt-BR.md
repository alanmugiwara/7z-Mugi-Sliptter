# :hammer: Funcionalidades do Projeto

- `Funcionalidade 1`: Dividir arquivos e pastas em arquivos de 1980MB
- `Funcionalidade 2`: Duas versões (uma apenas divide, outra divide e faz um nível de compressão maior)
- `Funcionalidade 3`: Fácil de fazer backup/compartilhar seus arquivos divididos no telegram grátis (o telegram grátis suporta 2GB para cada arquivo)
- `Funcionalidade 4`: Automação - basta arrastar arquivos e/ou pastas no script .bat e esperar seu PC fazer o trabalho pesado.
